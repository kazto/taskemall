# Task'em all

This is CLI for task management, logging, planning.

# install

write later...

# how to use

## initalize

```
taskemall init
```

create database in `$XDG_DATA_HOME/taskemall/` 

## check in / swap task / check out

```
taskemall check <TASK NAME> <CHECK TIME>
```

## view log (daily sequential)

```
taskemall log seq
```

## view log (daily summary)

```
taskemall log sum
```

## plan task (oneshot)

```
taskemall plan <TASK NAME> <PLAN DATETIME>
```

## plan task (recurring)

```
taskemall plan rec <TASK NAME> <PLAN DATETIME>
```
